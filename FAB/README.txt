Description of files included in XMP16-01-FAB.zip:

./README.txt    This File

./Gerber/   This directory contains the following gerber files for the design.

XMP16-01-BOT.pho   Bottom Copper Layer
XMP16-01-GND.pho   Ground Copper Layer
XMP16-01-NC.pho   NC Drill Layer
XMP16-01-PWR.pho   Power Plane Copper Layer
XMP16-01-SMT.pho   Solder Mask Top Layer
XMP16-01-SPT.pho   Solder Paste Top Layer
XMP16-01-SST.pho   Silkscreen Top Layer
XMP16-01-TOP.pho   Top Copper Layer

./Drill/   This directory contains the following drill data files for the design.

XMP16-01-NC.drl    NC Drill File (Excellon Format, no zero suppression, absolute coordinates)
XMP16-01-NC.rpt   NC Drill Report (Sizes & Quantities)

./Net/   This directory contains the netlist files for equivalence checking and bare board test.

XMP16-01-NT.net    Netlist

./XY/   This directory contains the following drill data files for the design.

XMP16-01-XY.pos   XY Component Position Data (Raw Data)
XMP16-01-XY.pos    XY Component Position Data (Excel Format)

./BOM/   This directory contains the bill of materials.

BOM.xlsx    Bill of materials (Excel Format)
