Description of files included in XK-1A-RevA-FAB.zip:

./README.txt        This file.

./BOM/      This directory contains the bill of materials and component position files for fabrication.

XK-1A-RevA-BOM.xls   Bill Of Materials
XK-1A-RevA-XY.xls    Component Position Data

./Drill/        Contains the following drill data files for the design.

XK-1A-RevA-NCD.drl     NC Drill File (Excellon Format, metric (000.000).
XK-1A-RevA-NCD.rep     NC Drill Report (Sizes & Quantities)

./Gerber/       Contains the following gerber files for the design in RS-274-X metric (000.000) format.
      
XK-1A-RevA-L1.pho     PCB Copper Layer 1 (where n = layer number, 01=Top Side)
XK-1A-RevA-L2.pho     PCB Copper Layer 2
XK-1A-RevA-SST.pho    Silkscreen Top
XK-1A-RevA-SSB.pho    Silkscreen Bottom  
XK-1A-RevA-SMT.pho    Solder Mask Top
XK-1A-RevA-SMB.pho    Solder Mask Bottom
XK-1A-RevA-SPT.pho    Solder Paste Top
XK-1A-RevA-SPB.pho    Solder Paste Bottom
XK-1A-RevA-DRL.pho    Drill Diagram

XK-1A-RevA-GBX.pdf    PDF of gerber data.
XK-1A-RevA-Assy.pdf   PDF of assembly diagram.
